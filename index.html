How to Use Filters in Eurostat URLs

To customize your data request, add filter parameters to the API URL. You can specify multiple values for a filter by repeating the parameter (e.g., &geo=DE&geo=FR for Germany and France). Common filters include:

  • time: Year or period (e.g., time=2023)
  • sex: Gender (e.g., sex=M for male, sex=F for female)
  • age: Age group (e.g., age=Y15-24)

Combine filters to refine your query and get exactly the data you need for your analysis or visualization.
