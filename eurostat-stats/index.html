Eurostat API Documentation

How to Use Filters in Eurostat URLs

Customize your data request by adding filter parameters to the API URL. Repeat a parameter for multiple values (e.g., &geo=DE&geo=FR for Germany and France). Common filters:

  • geo — Country code (e.g., geo=DE, geo=FR)
  • time — Year or period (e.g., time=2023)
  • unit — Unit of measurement (e.g., unit=NR)
  • sex — Gender (e.g., sex=M for male)
  • age — Age group (e.g., age=Y15-24)

Combine filters to refine your query and get exactly the data you need.
